Measurement-System

This is a template Python project with Sphinx documentation. Measurement-System based on RaspberryPi 5 hardware.

  • Measurements: DAQ-Hats (Digilent MCC Series) and RPi IOs

  • Visualization: ComVisu

Pre-Requisites

  • Hardware

    • RaspberryPi 5

    • Digilent MCC HATs

      • currently supported HATs: MCC118, MCC134

    • HX711 weight scales input module

  • Software

    • Installed the shared library and tools for MCC DAQ Hat

      • Summary

        cd ~
git clone https://github.com/mccdaq/daqhats.git
cd ~/daqhats
sudo ./install.sh

      • For more details, checkout https://mccdaq.github.io/daqhats/install.html

  • Toolchain

    • Install Hatch as python manager:

      • Python/Pip already there: pip install hatch --break-system-packages

      • Set global hatch config: hatch config set dirs.env.virtual .hatch

    • Notes

      • On Windows system, the lgpio library is not installed. Building docs and other features not available.

Install

Use Hatch to install dependencies:

hatch env create
